# Weather Dashboard

![Mockup Screenshot](assets/images/screenShot.png)

## Purpose

Using Open Weather API create a Weather Dashboard, when you select a city get the current weather and the 5 days forecast.

## Built With

- Javascript
- HTML & CSS
- Bootstrap 5
- localStorages
- Open Weather API
- Google Place API
- Momentjs
